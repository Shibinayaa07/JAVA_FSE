# Spring Data JPA Handson 1

Separate solution for `1. spring-data-jpa-handson.docx`.

Covered:
- Spring Boot `orm-learn` setup.
- Country entity, repository, and service.
- `findById`, `save`, `deleteById`, and partial country-name query.
- Notes/examples for Hibernate XML, Hibernate annotations, and JPA vs Hibernate vs Spring Data JPA are represented by the same entity/repository/service comparison in code.

Run:

```powershell
mvn -pl deepskilling/week_2/spring_data_jpa_handson_1 test
```
